# LA Crime Data Enhancement Pipeline 🚔

This project transforms raw LAPD crime data (2020-2024) into an analytically ready dataset. 

### 🌟 Key Features
- **MO Code Translation:** Converted 4-digit PDF codes into human-readable text.
- **Severity Scoring:** Developed a unique prime-number weighting system to rank crime seriousness.
- **Optimized Performance:** Processed 1,000,000+ rows using vectorized NumPy operations.

### 🛠️ Tech Stack
- Python (Pandas, NumPy)
- PDFPlumber and requests (for data scraping)
- Jupyter Notebooks

### 📊 Data Source
The processed data is hosted on Kaggle
https://www.kaggle.com/datasets/omershriky/los-angeles-crime-data-for-analysis-2020-2024
